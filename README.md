# Darkmail
You can visit the website: https://darkmail.kesug.com

## 📌 About The Project

Darkmail is a temporary email web application developed with PHP. It allows users to quickly create disposable email addresses and receive messages without registration. The purpose of this project is to demonstrate how temporary email services work and to provide a simple learning example for developers who want to explore email handling and mailbox systems using PHP.

This project is inspired by temporary mail services such as YOPmail and is designed for educational and experimental purposes.

---

## 🚀 Features

* Temporary email address creation
* No registration required
* Receive emails instantly
* Simple inbox interface
* Lightweight and fast system
* Easy-to-understand PHP source code

---

## 🧠 How It Works

Darkmail uses a simple temporary mailbox system:

1. The user enters or generates a temporary email address.
2. The system checks the mailbox associated with that address.
3. Incoming emails are displayed in the inbox panel.
4. Users can read messages without creating an account.

This structure demonstrates the core logic behind disposable email services.
